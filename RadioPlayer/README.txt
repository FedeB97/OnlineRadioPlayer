OnlineRadioPlayer - Example project
Package: com.onlineradioplayer.app

How to use:
1. Open this folder in Android Studio (Open an existing project).
2. Let Gradle sync. You may need Android Gradle Plugin compatible versions.
3. Build -> Build APK(s) to generate the debug APK.
4. Install on your device.

Notes:
- This is a minimal example intended for testing functionality (play/pause/next/prev) with ExoPlayer.
- The sample stream URLs are public examples; some may not always be reachable.
